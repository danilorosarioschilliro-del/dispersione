☁️ Configura Google Drive Condiviso
Come funziona: installi un piccolo script gratuito sul Drive della scuola. Ogni modifica viene salvata automaticamente. Tutti i coordinatori condividono gli stessi dati. Nessun dato viene mai perso.
Installazione — 10 minuti una volta sola
1. Scarica il file IIS Ramacca-Palagonia_Backend.gs (pulsante qui sotto)
2. Apri Google Drive con l'account scolastico → Crea cartella IIS Ramacca-Palagonia
3. Dentro la cartella: + Nuovo → Altro → Google Apps Script
4. Cancella tutto il codice e incolla il contenuto del file .gs scaricato
5. Alla riga var SECRET_KEY = 'ps_scuola_2026' — cambia la chiave con una tua
6. Salva → Distribuisci → Nuova distribuzione
7. Tipo: App web · Esegui come: Io · Chi accede: Chiunque → Distribuisci
8. Autorizza → copia l'URL → incollalo qui sotto
⚠️ La chiave protegge i dati — annotala e non condividerla.
